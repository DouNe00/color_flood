◆ processNeighbours()

processNeighbours ( game *  g,
int  x,
int  y,
int *  adjMatrix 
)

Modifie la matrice d'adjacence.

On choisit une case de la grille, on obtient son label et le label d'une case voisine : ces deux nombres sont les coordonnées dans la matrice d'une case où on inscrit 1. On procède de même pour chaque case voisine.

Les autres cases sont laissées à 0.

Parameters
g: la partie
x: ligne de la case choisie
y: colonne de la case choisie
adjMatrix: la matrice d'adjacence à modifier
Returns
La matriche d'adjacence modifiée
